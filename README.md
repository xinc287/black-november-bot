# Streamlit Trading Bot - Skeleton

Bu proje, kullanıcının istediği çok katmanlı analizlerin başlangıç iskeletini sağlar.

## Kullanım

1. Sanal ortam oluşturun

```bash
python -m venv .venv
source .venv/bin/activate    # Linux/Mac
.\.venv\Scripts\activate   # Windows
pip install -r requirements.txt
streamlit run streamlit_trading_bot.py
```

## İçerik
- streamlit_trading_bot.py - Ana uygulama
- requirements.txt - Gereken paketler

Geliştirme notu: ML modelleri, on-chain veriler ve gelişmiş backtest motoru için ayrı modüller ekleyin.
